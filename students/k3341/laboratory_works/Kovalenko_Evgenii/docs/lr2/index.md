# Лабораторная работа №2
## Система управления автогонками

### Описание проекта
Веб-приложение для организации автогонок с системой регистрации гонщиков, проведения соревнований и комментирования.

### Реализованные функции
- Регистрация и аутентификация пользователей
- Создание профиля гонщика
- Просмотр и регистрация на гонки
- Система комментариев с рейтингом
- Админ-панель для управления результатами
- Пагинация и поиск по гонкам
- Адаптивный дизайн с Bootstrap
- Разделение прав доступа

### Технологии
- Django 3.2
- Bootstrap 5
- SQLite
- Django Filter

### Запуск проекта
1. Активировать виртуальное окружение (подробнее [здесь](https://docs.google.com/document/d/1HAJibz51G8bY8tLZB7LFWUZxtnSznRUxwUhE4PsSqys/edit?tab=t.0#heading=h.czf688pt9dum))
2. Установить зависимости: `pip install django django-filter pillow`
3. Применить миграции: `python manage.py migrate`
4. Запустить сервер: `python manage.py runserver`
